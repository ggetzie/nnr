Markdown Help

*italics* = italics

**bold** = bold

[This is a link](http://thisisalink.com "link title") = This is a link

Use two spaces at the end of a line for a line break.

Use a blank line between paragraphs for a paragraph break.

#Heading1 = Heading level 1
##Heading2 = Heading level 2
###Heading3 = Heading level 3
                
Ordered list: 
1. First item 
2. Second item
3. Third item 
                
Unordered list: 
- First item 
- Second item
- Third item
